## Acknowledgement

`oh-novel-codex` was built with direct inspiration from and repeated reference to
[`oh-my-codex`](https://github.com/Yeachan-Heo/oh-my-codex) by Yeachan Heo.

This repository does **not** claim to be the same project. ONX is a separate,
fiction-focused workflow layer that borrows ideas from OMX's orchestration,
verification, packaging, and documentation discipline while adapting them to
long-form novel and rewrite workflows.
