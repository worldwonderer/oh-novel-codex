# Revision pass

Default priorities:
- fix P0 issues first
- then fix P1 issues
- keep the draft's strongest pages intact

Focus on:
- hook carry-through
- middle-act momentum
- side-character depth
- ending killshot protection
- removing summary drift
- replacing confession-dump truth delivery with more believable revelation paths
