---
name: remix-depth
description: Review a rewrite draft for structural originality. Use when a fiction rewrite may still feel like a rename-retell, a beat-for-beat expansion, or a scene-for-scene shadow version of the source.
---

# Remix depth

Check:
- conflict carrier changes
- pressure venue changes
- truth path changes
- punishment path changes
- ending landing changes

Fail the pass if:
- the draft still maps too neatly onto the source beat order
- the one-sentence synopsis is too close to the source synopsis
- the rewrite feels like the same machine with only renamed nouns
